# Poem-speak

Simple project using speech-to-text

## Technologies
This project was made using HTML, CSS, JavaScript, the Web Speech API

## How to use
Press spacebar and speak the poem

The lamp once out by Natsume Soseki\
The lamp once out\
Cool stars enter\
The window frame\
